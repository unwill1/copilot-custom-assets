# Copilot Custom Assets

This repository contains exported GitHub Copilot customization assets:

- prompts/
- skills/
- mcp/
- copilot-instructions.md
- mcp.template.json

## Usage

1. Copy the required files back to your local Copilot folders.
2. Rename mcp.template.json to mcp.json for local use.
3. Fill in the local paths and device-specific values in mcp.json.

## Notes

- mcp.template.json is sanitized for sharing.
- mcp.local.json is intentionally excluded from git.
